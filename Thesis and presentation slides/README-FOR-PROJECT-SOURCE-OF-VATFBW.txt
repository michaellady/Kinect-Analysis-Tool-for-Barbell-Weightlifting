README FOR PROJECT SOURCE OF VATFBW

CollectorNew, Analyser and Generator folders are the VS2012 project folder of each module of the application. The source files are located in the sub-directories, for example, /Analyser/Analyser for Analyser.

Executables folder contains complied executables of each module, which is ready to use.
